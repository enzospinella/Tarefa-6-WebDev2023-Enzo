CSS - Conceitos

É o estilo da página, o que modifica o tamanho da font dos textos, a imagem do fundo da página, entre outras coisas.

Na aula, aprendemos sobre algumas das principais tags de estilo do css, os seletores, como conectar o arquivo html com o arquivo css e, por fim, sobre o box model, um tipo de organização de layout de cada elemento html.
